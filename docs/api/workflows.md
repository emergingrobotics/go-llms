## Package Information

- **Import Path**: `github.com/lexlapax/go-llms/pkg/agent/workflow`
- **Category**: Agent Framework
- **Stability**: Stable (v0.3.x)

## Overview

The Workflows package enables composition of complex agent behaviors through declarative workflow definitions. It supports various execution patterns and provides robust error handling.

Key features:
- Sequential execution pipelines
- Parallel task execution
- Conditional branching
- Loop constructs
- Error handling and recovery
- State management across steps
