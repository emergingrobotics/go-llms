## Detailed Fix Requirements

### Provider Examples Fix Pattern:
```go
// OLD (incorrect)
provider := provider.NewOpenAIProvider(
    domain.NewOpenAIOrganizationOption("org-123"),
)

// NEW (correct)
provider := provider.NewOpenAIProvider(key, model,
    domain.NewOpenAIOrganizationOption("org-123"),
)
```

### Tool Examples Fix Pattern:
```go
// OLD (incorrect)
tool := tools.Tools.Get("file_read")
result, err := tool.Execute(ctx, map[string]interface{}{
    Path: "/path/to/file",
})

// NEW (correct)
tool, err := tools.Tools.Get("file_read")
result, err := tool.Execute(ctx, map[string]interface{}{
    "path": "/path/to/file",
})
```

### Import Path Fixes:
```go
// Correct imports
import (
    "github.com/lexlapax/go-llms/pkg/llm/provider"
    "github.com/lexlapax/go-llms/pkg/llm/domain"
    "github.com/lexlapax/go-llms/pkg/agent/tools"
    "github.com/lexlapax/go-llms/pkg/agent/builtins/tools/file"
    "github.com/lexlapax/go-llms/pkg/schema"
)
```
