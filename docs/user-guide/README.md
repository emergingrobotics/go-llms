## 📦 What's Included

### Built-in Tools (32+)
- **Web**: Search, fetch, scrape, API client, GraphQL
- **Files**: Read, write, search, manage
- **Data**: JSON, CSV, XML processing
- **System**: Commands, environment, system info
- **DateTime**: Parse, format, calculate, timezone conversion
- **Feeds**: RSS/Atom processing
- **Math**: Calculator with natural language

### Providers
- OpenAI (GPT-4o, GPT-4o-mini)
- Anthropic (Claude 3.5 Sonnet, Opus, Haiku)
- Google Gemini (Gemini 2.0 Flash Lite)
- OpenAI-compatible (Ollama, OpenRouter, custom)

### Workflow Patterns
- Sequential processing
- Parallel execution
- Conditional branching
- Loops and iteration

## 🚦 Prerequisites

- Go 1.21 or later
- Basic Go knowledge
- API keys for LLM providers (some examples work without)

## 💡 Tips

1. **Start small** - Begin with simple examples
2. **Use the playground** - Try things in the examples directory
3. **Read error messages** - They're designed to be helpful
4. **Check types** - Go's type system helps catch mistakes
5. **Ask for help** - Open issues for questions
